1/13 6:13 でんじろうに起こされ起床。今日もぷりぷりしていてかわいい

①リモート上の差分を取得する
git pull
もしくは
git fetch
git merge origin/ブランチ名

②ブランチ名を確認するためのコマンド
git branch

③変更されたファイルの確認
git status

④変更を登録する（コミットするために必要な工程）
git add ファイル名

⑤登録したファイルをコミットする
git commit -m 'コミットメッセージ'

⑥リモートとローカルのファイルの差分を抽出する
git diff ファイル名

⑦コミット履歴を確認する
git log

⑦リモートにプッシュ
git push origin HEAD
